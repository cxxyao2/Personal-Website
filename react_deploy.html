How to Deploy Your React 17 App to Heroku

Heroku

We need 4 steps for deploy your react on heroku. Your local machine, at the root folder of your project, terminal
  • google keywords: heroku buildpack for create-react-app, find a necessary package located in https://github.com/mars/...,copy the link
  • heroku login
  • heroku create --buildpack https://github.com/mars/...
  • git push heroku main
  • config environment variables: 
    $heroku config:set API_URL=https://xxx.xxx.xxx.xxx:5000/api
                Read the varaible in react:
            const apiUrl = process.env.API_URL;
  • Others commands:
    • npx create-react-app newProjectName
    • heroku open
    • heroku logs
Deployment is more easy than expected, thanking to many contributors.
